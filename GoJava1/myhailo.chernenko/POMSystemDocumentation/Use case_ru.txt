1. Пользователь запускает программу, вводит команду "добавить транзакцию" и получает возможность вводить транзакции 
	(данные о финансовой операции: сумму затрат (или дохода) по определенной статье и определенному проекту).
2. Пользователь запускает программу, вводит команду "показать" - видит список всех введенных ранее транзакций
3. Пользователь запускает программу, вводит команду "показать все проекты" - получает список проектов 
4. Пользователь запускает программу, вводит команду "показать доходы" - получает табличку с прибылью по проектам 
	(считаем сумму всех транзакций по каждому проекту)
5. ...
