Описание проекта медицинской системы.

Задача проекта состоит в автоматизации процесса по организации медицинских услуг лечебно-профилактическими учреждениями(ЛПУ).
Система будет выступать связующим звеном между ЛПУ и их клиентами - страховыми компаниями и клиентами физическими лицами. 

Стороны, которые будут использовать систему:

1. ЛПУ
Клиники должны иметь возможность с помощью разрабатываемой системы вносить информацию о своем учреждении, список принимающих врачей с описаниями и
направлениями работы, а также расписание приема. Планируется реализовать возможность записи на любую из процедур, которые предоставляют клинику. ЛПУ 
смогут предоставлять информацию о ценах на все процедуры, которые захотят показывать, для разных клиентов уровень цен может быть разным (в зависимости
от заключенного договора со страховой компанией или персональной программы скидок). В идеале моя система должна интегрироваться с действующей в клинике
системой и информация о графике врачей/запись должна происходить напрямую.

2. Клиенты
Клиентам должна предоставляться информация о свободном времени на когда возможно будет организовать желаемую услугу, а также функционал для отправки заявки на прием.
Заявки могут приниматься ЛПУ как в автоматическом режиме, так и подтверждатсья сотрудником клиники вручную. Клиенты смогут выбирать клинику/врача/процедуру по желаемому
направлению, территориальному расположению и уровню цен.

3. Администрация медицинской системы
Администратор(условно) со стороны систмы должен иметь возможность заводить новые карточки ЛПУ, назначать администраторов для них. Также администратор должен поддерживть
список медицинских направлений по которым в системе предлагаются услуги.

Идеи возможностей для расширения системы:
1. Реализация рабочего места для клиник в облаке, либо сделать модуль для локальной работы ЛПУ с CRM возможностями.
2. Интеграция со страховыми компаниями, либо загрузка у них покрытий для возможности самостоятельной записи на услуги застрахованным, проверки наличия покрытия со
стороны ЛПУ.
