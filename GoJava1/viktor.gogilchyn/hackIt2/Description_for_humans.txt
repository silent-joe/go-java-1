1. Мир игры состоит из множества Систем, которые производят Ресурсы. Система состоит из двух частей: Аппаратная часть и Програмная часть. 
1.1. Аппаратная часть определяется Характеристиками, которые учитываюся при установке и запуске Программы.
1.2. Программы, составляющие программную часть, имеют  требования (к Характеристикам Аппаратной части системы) и версию, которая определяет еффективность Программы. 

2. Основные действующие лица игры - Персонажи. Персонажи умеют взаимодействовать с любым игровым объектом - Вещами, Системами, их частями, другими Персонажами.
2.1. Персонаж может управлятся человеком или AI.
2.1. Персонаж в игре может имуществом Владеть или Распоряжаться. 
2.1.1. Имущество, находящееся во владении персонажа можно продать либо создать из имеющихся частей другое имущество. Никто из иных Персонажей не может изменить состояние владеемого имущества (TODO переписать!)
2.1.2. Имущество, находящееся в Распоряжении персонажа, дает Персонажу совершать с им любые действия, не являющимися имущественными. 
2.2. Персонаж во время игры может пытаться выполнить любые Действия. Если для полноценного совершения Действия недостаточно Ресурсов и/или Вещей, Персонаж об этом уведомляется. 

3. Основа взаимодействия в игре - Действия. С помощью Действия инициатор (Персонаж) сообщает миру о своем намерении, и мир соответственно реагирует. 
3.1. Действие всегда создает Результат, который обрабатывается Миром игры. Мир игры изменяет свое состояние только при получении Результата. 
3.2. Для совершения Действия у Персонажа должен быть развит Навык, соответствующий Действию. От уровня Навыка зависит результат Действия. 
3.3. Для каждого действия определен набор необходимых Ресурсов и/или Вещей, которые нужны для совершения Действия. При недостатке Ресурсов и/или Вещей Персонаж, инициировавший Действие, получит уведомление. 
3.4. Количество Действий, запущенных Персонажем, лимитировано соответствующим Навыком. Практически все Действия во время выполнения блокируют определенную часть Характеристик Аппаратной части, которая используется в Действии. 
3.5. Действие не может быть запущено, если недостаточно свободных Характеристик Аппаратной части.
3.6. Действия в игре не мгновенные. Время, неоходимое для совершения Действия, зависит от Навыка, используемых Вещей и/или Программ.
3.7. Действие может быть прервано до получения Результата, инициатором или другим Персонажем, которому известно про совершаемое Действие. 

4. Ресурсы. Для совершения многих (не не всех) Действий нужны ресурсы. Путей получения Ресурсов несколько: 1. Покупка в Магазине. 2. Создание 3. Составить из частей. Виды Ресурсов:
4.1.1 Деньги. Зарабатываются оказанием услуг (Задания), действиям в виде Наемника, после успешный выполнений некоторых Действий (например взлом банка ^_^ ). Тратятся в Магазине.
4.1.2 Расходуемые: ФайлыДанных, нужные для улучшения Программ. Запчасти, нужные для улучшения Аппаратной части. Можно получить, использовав определенное Действие, при наличии Системы во владении. 
4.1.3 Нерасходуемые: Программы, нужные для совершения большинства Действий, что касаются Систем. Инструменты, нужные для улучшения Аппаратной части с помощью Запчастей. Покупаются. 
4.2. Имущество. Всё Имущество в игре принадлежит кому-то из Персонажей. 
4.3. Программы. Предназначены для выполнения многих Действий. Может быть в двух состояниях - дистрибутива (не активна) или установленная (активна). Покупаются и разрабатываются. 
4.4. Вещи. Аппаратное обеспечение, не установленное в Систему; Устройства, специализированы для выполнения малого количества Действий (аналог Программы, но намного дороже и еффективней, не занимает ресурсы системы). Покупаются и собираются из Запчастей. 
4.5. Все покупки совершаются в игровом Магазине. 

5. Процесс игры. Игра в реальном времени, ходов нет. На любое Действие требуется определенное время. Состояние игры изменяется каждый раз при получении Результата. 
5.1. Состояние игры - совокупность состояния Систем, Имущества, Персонажей.
