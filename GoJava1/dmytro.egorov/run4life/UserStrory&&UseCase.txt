Назавание проекта:
run4life

Суть проекта:
Создать веб-сайт позволяющий составлять персональные программы тренировок для подготовки к забегам(5км, 10км, 21.0975км, 42,195км). 
Сайт содержит также новости и календарь с предстоящими событиями и ссылками на регистрацию.

Актеры:
Админ
Модератор
Пользователь
Посетитель

User stroies:
Пользователь может создать 1 персональных план.
Пользователь может удалить персональных план.
Пользователь может редактировать персональных план.
Пользователь может редактировать персональные данные.
Пользователь может вести журнал пробежек, по которому будет строиться статистика.
Пользователь может добавлять в друзья других учасников проекта.
Пользователь может обмениваться сообщения с друзьями.
Пользователь может сравнивать свою статистику со своими друзьями или гуру проекта.
Пользователь моет просматривать новости.
Пользователь моет просматривать ивенты.

Админ может создавать нового модератора.
Админ может просматривать всех модераторов.
Админ может создавать новости.
Админ может просматривать все новости.
Админ может создавать ивенты.
Админ может просматривать все ивенты.
Админ может просматривать всех пользователей.
Админ может блокировать пользователей.

Посетитель может просматривать новости.
Посетитель может просматривать ивенты.
Посетитель может просматривать заригестрироваться на сайте.

Модератор может создавать новости.
Модератор может просматривать все новости.
Модератор может создавать ивенты.
Модератор может просматривать все ивенты.

//******************************************************************************************************************************************************************************************************//
Имя сценария
Создание персонального плана.

Цель
Создать персоналый план для пользователя основываясь на вводимых данных.

Актеры
Пользователь.

Предварительные условия
Пользователь должен выполнить вход на сайт.
Активаторы
Пользователь нажимает кнопку "Создать персональный план".
Порядок событий
Пользователь выбирает необходимые атрибуты(дистанция, дата забега, количество тренировок, уровень подготовки, результаты предыдущего забега, желаемое время);
Система выполняет расчет плана тренировок.
//******************************************************************************************************************************************************************************************************//
Имя сценария
Изменение персонального плана.

Цель
Вносить изменения в персоналый план пользователем.

Актеры
Пользователь.

Предварительные условия
Пользователь должен выполнить вход на сайт.
Пользователь должен иметь план тренировок.
Активаторы
Пользователь нажимает кнопку "Редактировать персональный план".
Порядок событий
Пользователь изменяет необходимые атрибуты(дистанция, дата забега, количество тренировок, уровень подготовки, результаты предыдущего забега, желаемое время);
Система выполняет расчет плана тренировок.
