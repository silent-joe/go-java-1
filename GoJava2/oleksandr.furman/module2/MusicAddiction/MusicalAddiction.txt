MusicAddiction – рекомендательная сеть для любителей слушать и обсуждать музыку.

1. Как гость я могу зарегистрироваться в системе для получения пользовательской учетной записи и последующей работы. Нужен проверенный e-mail и выбранные пользователем имя и пароль.
1) пользователь не может ввести пароль меньше 6 символов
2) пользователь не может ввести имя меньше 3 и больше 20 символов
3) пользователь должен иметь уникальное имя в системе
4) пользователь должен пройти бот-защиту.
5) после регистрации пользователь должен получить имейл для активации своей учетной записи
6) пользователь не может войти в систему, если учетная запись не была активирована
7) при успешном входе система приветствует пользователя текстом «Добро пожаловать, <имя пользователя>»

2. Как гость я могу войти в систему под ранее созданной учетной записью, для последующей работы
1) пользователь должен ввести правильное сочетание логин/пароль в пределах 5 попыток после чего должен пройти бот-защиту.
2) пользователь не может ввести имя меньше 3 и больше 20 символов.
3) если учетная запись не была активирована система уведомит пользователя текстом «Учетная запись <имя пользователя> не активирована, выполните инструкции описанные в письме активизации учетной записи, отправленном по адресу <имейл  пользователя>»

3. Как пользователь я могу искать музыкальные записи в системе.
1) пользователь может искать музыкальные записи по ключевым словам.
2) пользователь может смотреть каталог записей по категориям (исполнитель/год выпуска/жанр/альбом/….)

4. Как пользователь я могу прослушивать музыкальные записи в системе.
1) пользователь может прослушать любую музыкальную запись, загруженную в систему.
2) пользователь может составлять и сохранять свои плейлисты.

5. Как пользователь я могу хранить музыкальные записи в системе.
1) пользователь может создать альбом или папку для хранения своих музыкальных записей.
2) пользователь может выбрать куда загрузить музыкальную запись.
3) пользователь может загрузить музыкальную запись с локального диска в систему.

6.Как пользователь я могу комментировать музыкальные записи в системе.
1) пользователь может оставить текстовый комментарий, длиной до 250 символов, любой музыкальной записи в системе.
2) пользователь может удалить свой комментарий. 

7. Как пользователь я могу оценивать музыкальные записи в системе.
1) пользователь может поставить оценку от 1 до 10 любой музыкальной записи в системе.

 8. Как администратор я могу управлять записями пользователей, так чтобы контент сайта был легальным.
1)	администратор может удалить любую музыкальную запись/альбом/комментарий.
2)	администратор может удалить учетную запись пользователя.

 9. Как пользователь я должен иметь возможность восстановить забытый пароль. 
1) пользователь должен ввести логин/e-mail, после чего получить пароль по имейлу.

10. Как пользователь я могу удалить свою учетную запись и перестать быть пользователем системы.
1) Пользователь может удалить свою учетную запись.

 11. Как пользователь я могу изменить данные своей учетной записи.
1) пользователь может изменить пароль.
2) пользователь получить имейл с подтверждением изменения пароля и подтвердить изменение. 

 12. Как пользователь я могу сделать некоторые поля своей учетной записи видимыми для других пользователей.
1) пользователь имеет возможность выбрать отображать/скрывать подробную информацию о себе.
